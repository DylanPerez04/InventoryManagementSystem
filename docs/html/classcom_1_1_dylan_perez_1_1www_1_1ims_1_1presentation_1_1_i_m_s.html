Detailed Description

IMS is the base class for all Inventory Management System queries used by a client to facilitate inventory stock and sales analytics for all in-store and online transactions with their business. An IMS object encapsulates the means by which a client can add, remove, and navigate through their available–and unavailable–selection of products. In addition, the IMS object directly handles client-to-customer transactions which interact with the stock of the client's available selection, allowing the IMS object to provide analytical data regarding the client's sales.
